# 📈 Stock Trading Platform

A simple **Java Console-Based Stock Trading Platform** that allows users to view available stocks, buy stocks, sell stocks, and manage their investment portfolio.

---

## 🚀 Features

- 📊 View available market stocks
- 💰 Buy stocks
- 📉 Sell stocks
- 📁 View portfolio
- 🧮 Calculate total portfolio value
- 🖥️ Menu-driven console interface

---

## 🛠️ Technologies Used

- Java
- ArrayList
- Object-Oriented Programming (OOP)
- Scanner Class

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/Shub148/StudentGradeTracker.git

```

### 2. Navigate to the Project

```bash
cd StockTradingPlatform
```

### 3. Compile the Program

```bash
javac StockTradingPlatform.java
```

### 4. Run the Program

```bash
java StockTradingPlatform
```
